## 系統簡介

本系統是一個專為大學生設計的食譜管理 Web 應用。解決了使用者難以記錄零散食譜的問題，提供新增、查看與儲存功能，讓下廚變得更簡單有序。

---

## 技術棧

- **前端：** Jinja2
- **後端：** Python + Flask
- **資料庫：** SQLite
